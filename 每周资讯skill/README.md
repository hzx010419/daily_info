# 每周资讯Skill

基于DeepSeek AI的每周资讯整合工具，自动收集每日资讯文件，进行主题识别、AI融合、过滤、筛选、分类，生成每周资讯汇总文档。

## 功能特性

- **自动周范围计算**：自动计算本周（周一到周日）或指定日期所在周的时间范围
- **批量文件解析**：自动解析每日资讯Word文档，提取文章信息
- **智能内容过滤**：
  - 自动过滤**聚合新闻**（一条资讯包含多个不相关主题，如"恒大许家印案开庭、2026年电影票房破125亿 | 科股快报"）
  - 自动过滤**社会评论/外界误解分析类**内容（如"熬到40岁当主任成医界最大误解"类揭示职业误解的文章）
- **主题识别与AI融合**：
  - 识别同一题材的多篇文章
  - 调用DeepSeek API进行信息融合（保留并集、时间线整理、观点分块）
  - 生成新的整合摘要和标题
  - 汇总所有原文链接
- **小热点过滤**：自动过滤掉非常小的社会热点（如某地医药问题、某学校改制等）
- **MECE分类整理**：按照与每日资讯相同的MECE分类框架整理文章
- **Word文档输出**：格式与每日资讯保持一致，生成精美的每周资讯汇总文档

## 安装步骤

### 1. 克隆/复制项目

将整个`每周资讯skill`文件夹复制到目标机器。

### 2. 安装Python依赖

```bash
cd 每周资讯skill
pip install -r requirements.txt
```

### 3. 配置API密钥

编辑 `src/main.py`，修改API密钥：

```python
DEEPSEEK_API_KEY = "你的DeepSeek API密钥"
```

> 获取地址: https://platform.deepseek.com/

## 使用方法

### 命令行模式

```bash
# 处理本周资讯（默认）
python run.py

# 处理指定日期所在周的资讯
python run.py --date 2026-04-13

# 处理上周资讯
python run.py --last-week

# 指定每日资讯目录
python run.py --daily-dir "D:/每日资讯"

# 指定输出目录
python run.py --output "D:/每周资讯"

# 详细输出模式
python run.py --verbose
```

### Python调用模式

```python
import sys
sys.path.insert(0, '每周资讯skill/src')
from main import main
from datetime import datetime

# 处理本周资讯
success = main()
```

## 输入输出说明

### 输入

- **每日资讯文件**：从 `../每日资讯/` 目录读取 `每日资讯_YYYY-MM-DD.docx` 文件
- **日期范围**：自动识别文件名中的日期，筛选出本周范围内的文件

### 输出

- **输出文件**：`每周资讯/每周资讯_YYYYMMDD_YYYYMMDD.docx`
- **命名格式**：例如 `每周资讯_20260413_20260419.docx`
- **文档格式**：与每日资讯格式保持一致
  - 按MECE分类排序
  - 中文数字序号（一、二、三...）
  - 包含：标题、来源、时间、摘要、链接
  - 融合文章额外包含：多链接汇总、时间线、观点分块

## 处理流程

1. **收集文章**：扫描每日资讯目录，解析本周范围内的所有Word文档
2. **日期过滤**：只保留发布时间在本周范围内的文章
3. **过滤聚合新闻**：过滤掉一条资讯包含多个不相关主题的内容（如"恒大许家印案开庭、电影票房..."）
4. **过滤社会评论**：过滤掉外界误解分析、感慨类内容（如"熬到40岁当主任成医界最大误解"）
5. **主题识别与AI融合**：
   - 使用关键词Jaccard相似度找出相似文章
   - 使用Union-Find算法构建相似文章组
   - 对每个题材组，调用DeepSeek API进行信息融合
   - 融合内容包括：信息并集、时间线整理、观点分块
   - 生成新的整合摘要和标题，汇总所有原文链接
6. **分类整理**：按MECE分类框架排序整理
7. **生成文档**：输出格式统一的Word文档

## 过滤逻辑

### 聚合新闻过滤

以下特征的文章会被过滤：

- 标题包含多个主题词（用"|"、"、"、"和"等分隔）
- 摘要中包含多个转折词引出多个话题（"另外"、"与此同时"、"此外"等）
- 一条资讯像新闻简报一样汇总了多个不同新闻

### 社会评论过滤

以下特征的文章会被过滤：

- 揭示"外界对某职业/群体的误解"
- 分析"现实与想象的差距"
- 用"XX不是XX"、"XX并非XX"、"XX最大误解"的句式
- 大量感慨性语言，如"太难了"、"太卷了"、"熬出头"

### 主题融合逻辑

当多篇文章讨论同一题材时，AI会：

1. **信息并集**：保留所有文章中的有价值信息，不遗漏
2. **时间线整理**：如果涉及事件发展，按时间顺序整理叙事
3. **观点分块**：将不同角度/不同立场的内容分别整理
4. **生成摘要标题**：基于融合内容生成一个简洁准确的标题
5. **汇总链接**：收集所有原文链接

### 小热点过滤规则

自动过滤符合以下特征的文章：

- 某省/市/县/区的医院/学校/药店事件
- 地方院校改制、合并、调整
- 非全国重点高校的一般动态
- 缺乏全国性关键词（国务院、教育部、国家卫健委等）

## 常见问题

### Q: 如何调整相似文章识别敏感度？
A: 编辑 `src/ai_client.py`，修改 `SIMILARITY_THRESHOLD` 值（默认0.06）

### Q: 可以跳过过滤步骤吗？
A: 可以，注释掉 `main.py` 中 `filter_aggregated_news` 和 `filter_social_commentary` 的调用即可

### Q: 如何处理跨周的文章？
A: 程序会根据文章发布时间（从Word文档中解析）判断是否属于本周，而不是文件日期

### Q: 可以定时自动运行吗？
A: 可以，建议每周日晚上运行，使用Windows任务计划程序或Linux cron定时执行

```bash
# 示例：每周日23:00运行
0 23 * * 0 cd /path/to/每周资讯skill && python run.py
```

## 注意事项

1. **API调用费用**：融合功能会调用DeepSeek API进行文章融合和多维度判断，会产生一定费用
2. **文件编码**：确保每日资讯Word文档格式正确，否则可能解析失败
3. **日期格式**：每日资讯文件名必须包含 `YYYY-MM-DD` 格式的日期
